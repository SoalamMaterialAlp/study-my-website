Aim:-Practice of various Linux/Unix commands like man, cp, mv, In, rm, mkdir, rmdir etc.
Requriment:-Personal computer or laptop, Install Linux/Unix Operating System, Virtual machine – vmware, virtual box, server machines – hosting, cloud
Procedure:-
Install Linux/Unix Operating System:

1.First download the ISO image of the Linnux/Unix system you want to install from its official website .
2.Prepare a USB drive with at least 8 GB of storage.
3.Create a bootable USB using a tool such as rufus, Balena, Etcher or ventoy.
4.Restart your computer and open the boot menu (usually by pressing keys like F12, F9 during start up).
5.Select the USB drive as the boot option.
6.On the welcome screen, chose “Try or Install” for Linux or “Install for “ Unix.
7.Select your preferred language and keyboard layout.
8.Choose whether you want a normal installation (with most apps) or a minimal installation.
9.Allow installation of updates and third-party drivers if available.
10.Decide the installation type.

  • If you want Linux , Unix only choose erase disk and install.
  • If you want full control choose manual portioning.
  • If you want to keep windows and Linux/ Unix choose install alongside windows.

11.For manual partitioning create a root partition a swap partition and optionaly a home partition.
12.Select your time zone.
13.Create username computer name and password.
14.Begin the installation wait while files copied.
15.When the installation fishes restart the system.
16.Remove the USB drive when prompted.
17.The computer will now boot into Linux/Unix.
18.Log in with the username and password you created.
19.After logging in connect to the internet and checks for updates.
20.Install Linux/ Unix operatimg system successfully.
21.Install additional applications and drivers as needed.


Some Basic Linux/Unix Commands:-

File and Ditectory Folder Commands:-


command work
pwd the current directory path
ls List files and folders
cd Navigates to a specific folder
cd-- goes to step back
mkdir folder name Create new folder
rmdir folder name Delete empty file
cp source destination copy file
mv oldname new name File folder rename/move
unlink Delete file single file

Viewing and Editing Files:-


command work
Cat file name Display the content of a file
Less file name Show file content page by page
nano file name simple editor to edit a file
vim file name advanced text editor
touch file name creates an empty file

Users and permissions Commands:-


command work
whoami shows the current user name
who shows who is logged into the system
passwd Change your password
chmod 755 file name changes file permissions
chown user:group file changes the file owner

System Information Commands:-


command work
usname-a shows full system information
date shows current date and time
df-h checks disk space
free-h checks RAM memory usage
top shows running live
uptime shows how long the system has been running
