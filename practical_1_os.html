Practical No.-1

Aim:-Format and Install Windows Operating System.
Requriment:- Personal Computer or Laptop, windows installation USB Bootable drive backup of important files product key.

Formatting and installing the windows operating system involves erasing exiting data form the drive and setting up a new operating system. It is device when system become slow or corrupted.
Virus infection damages the operating system .A new version of windows is needed.
Procedure:
Step 1: Backup important data:
Copy all your important files to an external drive or cloud storage before starting.
Step 2: Insert installation media:
Plug in to bootable USB or insert the windows DVD. Restart your computer.
Step 3: Boot from installation media:
During startup press the required key F2, F12, ESC etc to open the boot menu. Select your USB or DVD drive.
Step 4: Windows setup screen:
Chose language time and currency format and keyboard layout. Click then Click Install Now.
Step 5: Enter Product Key:
Enter your windows product key or select I donâ€™t have a product key to continue.
Step 6: Select Installation Type:
Choose custom: Install windows only advanced.
Step 7: Format the drive:
Select the partition where windows will be installed.
Click format then Confirm.
Step 8: Install Windows:
Select the formatted partition and click next wait while windows copies files install features and updates.
Step 9: Initial Setup:
After restart set up region keyboard layout user Account and privacy setting.


Precautions:
  • Always backup your important files before formatting.

  • Download windows only from the official Microsoft websites.

  • Keep your computer connected to power during the installation.

Results:

Successfully formatted and hard disk and completed a clean installation of windows.
