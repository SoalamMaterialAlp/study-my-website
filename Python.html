Scripting Language Using Python



SYCRIPTING LANGUAGE (PYTHON) SYLLABUS

COURSE CONTENT

UNIT 1: Introduction, Variables and Data Types:

History, Features, setting up path, Installation and Working with Python, Basic Syntax, Understanding Python variables, Numeric data types, using string data type and string operations, Basic Operators, understanding coding blocks, defining list and list slicing, Other Data Types (Tuples, List, Dictionary, Set).

UNIT 2: Control Structures:

Conditional blocks using if, else and elif, for loops and iterations, while loops, Loop manipulation using continue, break.

UNIT 3: Functions, Modules, Packages and Exception Handling:

Organizing codes using functions, organizing projects into modules, Importing own module as well as external modules, Understanding Packages, Exception Handling.

UNIT 4: File I/O, Text Processing, Regular Expression:

Understanding read functions, Understanding write functions, Programming using file operations, Pattern matching and searching, Pattern searching using regex.

UNIT 5: Classes in Python:

Principles of Object –Oriented Programming, Concept of Classes, Class Variables, Creating Classes, Instances, Methods, File Organization.
