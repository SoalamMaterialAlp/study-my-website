Aim:-Program using expression evalution and precedence.
(a)-Write a program using unary and ternary operators.
(b)- Write a program using postfix and prefix increment operators.

Required:-

Personal computer or laptop, Install visual studio code with c compiler and extension OR Turbo C++


(a)-Write a program using unary and ternary operators. 

            
    C  
    #include 
int main(){
    //unary operators
    int a;    
    printf("Enter the number:");
    scanf("%d",&a); //address of(&)a
    printf("unary minus(-a):%d\n",-a);
    printf("Postfix increment:%d\n",a++); //postfix incrment
    printf("%d\n",a);
    printf("Prefix increment:%d\n",++a); //prefix increment
    printf("Postfix decrement:%d\n",a--); //postfix decrement
    printf("%d\n",a);
    printf("Prefix decrement:%d\n",--a); //prefix decrement
    printf("Logical not:%d\n", !(a%2 ==0));
    //tertiary operators
    int age;
    printf("Enter the age:");
    scanf("%d",&age);
    age>=18?printf("Adult"):printf("Not adult");
    return 0;
}

(b)- Write a program using postfix and prefix increment operators.
