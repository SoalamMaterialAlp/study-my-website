Question 1:-Explain data type in C?

A data type specifies the size and type of information a variable can store.

    1.Primitive Data Type
  • Primitive data type are the basic built-in types provided by the compiler.
  • The store simple values like numbers, decimals or characters.
  • It also known as primary data types

Types of Primitive Data Types
    1.(i):-int
  • int is a keyword in C. It is used to declare variables that store integer values.
  • integer is used to store whole numbers without decimals values.
  • It can store both negative and positive number.
  • In C language, %d is used to input and output integer values.
  • The size of int is usually 2 or 4 bytes according compiler.
  • The standard range for integer data types
    • for 4-byte (32-bit) int = -2147483648 to +2147483647
    • for 2-byte (16-bit) int = -32768 to +32767
  • Example: int a = 35;
    printf("A"%d\n",a);

    1.(ii):-float:
  • float keyword it is used to represent the floating point data types.
  • Floating data types are used to store decimal and fractional parts(number).
  • The size of float is 4 bytes (32 bits) in most system.
  • They can store botth positive and negative values.
  • Main floating data types are float,double and laong double.
  • Example: float price = 99.99;
    printf("Price:%2f\n",price);

    1.(iii):-double:
  • double is a primitive data types in C.
  • It is used to store decimal numbers doubles provides more precision then float.
  • It usually occupies 8 bytes of memory.
  • It can store very large and very small decimal values.
  • double is used when high accuracy is required.
  • Example: double pi = 3.141592;

    1.(iv):-char:
  • har is a keyword in C. It is used to declare varicables that store character.
  • Character data tyoes are used to store single character a single value in single Quotes('').
  • A character data types take upto 1 byte memory store.
  • It stores character using ASCII (American Standard Code for Information Interchange) values internally.
  • %c is used for input and output.
  • Example: char a = 'K';
    printf("character:%c",a);

    1.(v):-void:
  • void is a primitive data type in C.
  • It means no value or nothing.
  • void is mainly used with functions.
  • A function with void return type does not return any value.
  • It is used when no data is required or returned in a program.
  • void can also be used in functions parameters.
  • void main() indicates the function does not return a value (not recommended in Stanadard C).

Different Types of Pointer:-

  • A pointer that points to nothing.

    int a = 5;

    free(ptr);
