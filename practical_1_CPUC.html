Practical No.-1

Aim:-Installation of C compiler

Requirement :-Personal Computer or laptop ,Internet connection.

Procedure:-

Download Turbo C++

Turbo C ++ is an open source compiler available for free online. You can download it.

Step-1: Search for Download Turbo C++ on Google you can download Turbo C ++ for trusted/official websites for free figure.

Step-2: The download will start and after downloading open the Zip folder. You can find the zip folder in your download file in the file manager.

Step-3: Click on the setup.exe file inside the zip folder.

Step-4: Now the setup configuration window will open and the window installer will prepare for installation.

Step-5: Click the next button to start the Turbo C++ installation .DO not close the window before installation completes.

Step-6: Accept the license agreement by checking the checkbox in the dialog box.

Step-7: Installation for Turbo C++ will start.

Step-8: Click on finish button once the installation gets completed.

Step-9: A new window of Turbo C++ will appear on your screen. You can Start a new project.

Result:-

Now we successfully the installation complete. You can use the Turbo C++.
