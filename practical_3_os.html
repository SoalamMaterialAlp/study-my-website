Practical No.:- 3

Aim:-: Use Task Manager to monitor and terminate process. Understand process priorities.
Requirement:- Personal Computer or Laptop.
Procedure:-
Step 1:- Start Personal computer or laptop.
Step 2:- Go to task bar, click the right button given option and select the task manager click.
Step 3:-Open Task manager. You will see running apps and background running processes. Columns C.P.U. memory Disk Network etc.
Terminate Processes
Step 4:-Any Apps and background process are terminate select the app process and right click the End task option click.
Step 5:-Now Apps/process are stop.
Precautions:-
Do not terminate unknown system process.
Example:- csrss.exe killing critical system process can crash windows or log you out. If unsure, search online before ending task.
Results:- You will see the task manager then you system use (C.P.U., memory) high 70%-90% than you terminate the unusable apps/process than C.P.U. is good.
